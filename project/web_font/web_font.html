지금 보고계시는 폰트는 나눔 고딕폰트립니다. 구글 웹폰트를 css에서 @import 으로 불러왔습니다.
@import url('https://fonts.googleapis.com/css?family=Nanum+Gothic|Nanum+Myeongjo&display=swap&subset=korean');
